This web app tests the Collatz conjecture. Enter a number and press the button labeled 'Test'.

The Collatz conjecture is...

  1. Take any whole number
  2. If it is even: divide it by two to get a new number.
  3. If it is odd: triple it (multiply it by three) and add one to get a new number.
  4. Using the new number value repeat steps 2 and 3 until the number is 1.

Notes

  • The numbers 0 and 1 are not acceptable.
  • To test a new number overwrite the value that is in the text box and click the Test button again.
